Icons available in Touch UI applications

Add the prefix "ui-icon-" to the icon name to reference icon image.

For example, use the following to create a button with the action icon:

<a class="ui-btn ui-btn-icon-left ui-btn-inline ui-icon-action">action</a>

Use the "ui-icon-" + icon name in the Css Class property of an action to set the icon for that action. These icons are supported only in Touch UI.

Glyphs available in Touch UI and Desktop applications

Add the class "glyphicon glyphicon-" to the icon name to reference icon image.

For example, use the following to create a word with a glyph to the left:

<span class="glyphicon glyphicon-ok">Text</span>

Use the "glyphicon-" + icon name in the Css Class property of an action to set the icon for that action. Supported in Desktop and Touch UI.

These glyphs were included with the Bootstrap library for free by http://glyphicons.com/.
